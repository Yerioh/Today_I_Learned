# Markdown

- 일반 텍스트로 문서를 작성하는 간단한 방법
- 주로 개발자들이 텍스트와 코드를 작성해 문서화하기 위해 사용

## 1. 특징

- 작성된 Markdown 문서는 다른 프로그램에 의해 변환되어 출력됨
- Markdown 파일 확장자는 .md

## 2. 자주 사용하는 문법

[[마크다운 가이드]](https://www.markdownguide.org/basic-syntax/) 다양한 활용법 알아볼 것!! ✔
